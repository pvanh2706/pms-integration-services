2. Messages from TigerTMS to PMS 
 
The PMS interface should make available via a soap URL the following method call which takes a string parameter, and returns a string parameter – “SUCCESS” or “FAILED” . 
If “FAILED” is returned please extend the response with a failure reason for logging purposes. Eg. “FAILED – Invalid room number.” 
 
String SendMessageToExternalInterface (string Msg)  - General single input method. 
 
A simple C#  Example of the required WebMethod is as follows: 
 
namespace InterfaceMessageOut 
{ 
    /// <summary> 
    /// Summary description for Service1 
    /// </summary> 
    [WebService(Namespace = "http://CONNECTED GUESTSgenericinterface.org/")] 
    [WebServiceBinding(ConformsTo = WsiProfiles.BasicProfile1_1)] 
    [System.ComponentModel.ToolboxItem(false)] 
    // To allow this Web Service to be called from script, using ASP.NET AJAX, uncomment the following line.  
    // [System.Web.Script.Services.ScriptService] 
    public class InterfaceMessageOut : System.Web.Services.WebService 
    { 
        public string SendMessageToExternalInterface(string Msg) 
        { 
            string Response = "SUCCESS";             return Response; 
        } 
     } 
     
} 
 
 
 
Due to the fact that the WEB API is constantly evolving, please successfully acknowledge any messages which you do not implement in order for the communication to continue without any timeouts/retransmission attempts. 
 
 
 
The following types of messages will be sent to the PMS system 
 
Charge Records:  Different Types – CallRecord, Internet , Minibar and Video 
Room Status  
Wakeup Answer Status 
Request Bill 
Request Checkout 
Request Refresh 
Voicemail

If xml nodes are not used by the interfacing system eg. userdefine0 then TigerTMS will still pass the values but as a blank string eg. <userdefine0></userdefine0> or will not supply the nodes, the  external interface should cater for either scenario. 
 
The wsuserkey is a unique key used to identify the system, this should be configurable at the interfacing system
